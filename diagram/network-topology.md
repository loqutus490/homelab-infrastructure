# Network Topology

This document describes the network topology of the homelab infrastructure.

## Topology Overview
The network topology is structured in a hierarchical manner, consisting of core, distribution, and access layers.

### Core Layer
- High-speed backbone connecting all data center switches.
- Provides redundancy and high availability using routing protocols.

### Distribution Layer
- Aggregates traffic from the access layer switches.
- Implements policies for routing and quality of service (QoS).

### Access Layer
- Connects end devices such as computers, printer, and other peripherals.
- Provides VLAN segmentation for better management and security.
